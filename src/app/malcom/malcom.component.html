Malcolm is a powerful, easily deployable network traffic analysis tool suite for full packet capture artifacts (PCAP files) and Zeek logs and designed with the following goals in mind:
  • Easy to use
  • Powerful traffic analysis
  • Streamlined deployment
  • Secure communications
  • Permissive license
  • Expanding control systems visibility
