Cybersecurity Self Evaluation Tool

CSET is a self-contained software tool that runs on a desktop or laptop computer. CSET provides a systematic, disciplined, and repeatable approach for evaluating an organizationâ€™s security posture. CSET guides asset owners and operators through a step-by-step process to evaluate ICS and IT network security practices. CSET is offered at no cost and can be found on Github.

On June 30, 2021, CSET was updated to include a new module: Ransomware Readiness Assessment (RRA). The RRA is a self-assessment based on a tiered set of practices to help organizations better assess how well they are equipped to defend against and recover from a ransomware incident.
